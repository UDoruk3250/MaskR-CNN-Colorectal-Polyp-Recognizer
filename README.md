# MaskR-CNN-Colorectal-Polyp-Recognizer

This is a Mask R-CNN model trained to recognize colorectal polyps. The model is able to segment polyps in colored images.

### Setup

The installation process is fairly easy. It needs Python >=3.9 to work properly.

```bash  
pip install -r requirements.txt
```


### Usage

```bash  
python run.py [PATH_TO_IMAGE] -o [OUTPUT_FILE_NAME]
```

Output:

A new image file generated with the original image visually segmented with polygons, each with confidence scores.
